
\subsubsection{Les jetons}
\label{tokens}

De même que pour les architectes, nous ne pouvions pas modifier le fichier \code{token.h}, cependant, l'implémentation du type jeton n'est pas abstrait, il est donc possible de créer des jetons n'importe où dans le projet

Nous avons cependant décidé, comme pour les architectes, de créer au début de la partie \code{NUM\_TOKENS} jetons, puis de les manipuler par l'intermédiaire de pointeurs. Cela a l'avantage d'avoir un nombre maîtrisé de jetons tout au long de la partie.
